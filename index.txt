Desafio Técnico – React Native (Expo) 
Contexto 
Uma rede de varejo em expansão enfrenta dificuldades no controle de suas filiais e 
estoques. Atualmente, o controle é feito manualmente em planilhas Excel, que ficam 
rapidamente desatualizadas. Foi solicitado o desenvolvimento de um aplicativo móvel 
multiplataforma (Android/iOS) que centralize o cadastro das lojas físicas e de seus 
respectivos produtos. 
O sistema deve permitir: - Cadastro e listagem de lojas. - Cadastro e listagem de produtos 
vinculados a cada loja. 
�
�
Estrutura do Projeto 
O candidato deverá desenvolver o projeto utilizando React Native com Expo, seguindo boas 
práticas de arquitetura, design e usabilidade. 
Requisitos técnicos mínimos: - Expo SDK 54 ou superior - React 19 / React Native 0.81+ - 
TypeScript obrigatório - Navegação com Expo Router - UI com Gluestack UI. - Estado com 
Zustand, Jotai, Recoil ou Context API - Simulação de back-end com MirageJS ou Mock 
Service Worker (MSW) - Deve haver endpoints simulados para /stores e /products - Cada 
“loja” pode ter um array de “produtos” associados
